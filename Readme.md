## [M1S7] Ex 1 - Modelagem

### Descrição

Modelar tabelas "curso" e "professor"

1. Tabela curso deve conter ao menos:
- Colunas:
  - ID
  - Nome
  - Descrição
  - Professores
  - Carga Horária

2. Tabela professor deve conter ao menos:
- Colunas:
    - ID
    - Nome
    - Especialidade
